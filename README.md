### Description

Unity Editor Essentials (currently aimed for Unity 6000.3 and later)

### Installation

- Unity Package Manager: Install Package from GIT Url...
######
      https://github.com/unitycoder/UnityEditorEssentials.git
- Right Click on editor maintoolbar, Enable These buttons from Tools/... (you can move them around by holding Ctrl button and drag with mouse)


# Features

## MainToolbar Custom Buttons
- Build Settings (Build Profiles)
- Project Settings
